# Ritu IVF Call Sync App - Quick User Guide

## Must Have (Important)

- Phone must have **oDialer app installed**.
- **oDialer must be set as the default dialer app**.
- Internet must be ON.

Without this, call recording sync may not work correctly.

## First-Time Setup

1. Install the latest APK.
2. Open app -> `Settings`.
3. Enter:
   - `Device name`
   - `Device phone number`
4. Grant all required permissions:
   - Call logs
   - Contacts
   - Phone
   - Storage/Media

## Daily Use

- Open `Call Logs`.
- Tap `Load (last 100)` to refresh.
- Tap cloud icon for single call sync, or `Push to portal` for all visible calls.
- Synced calls show `Synced`.

## Auto Sync

- App auto-syncs about every **30 minutes**.
- Also runs when app is opened/resumed.
- Android may delay exact timing (battery/background rules).

## If data is not showing on portal

1. Check internet connection.
2. Confirm oDialer is default dialer.
3. Open app and tap `Push to portal`.
4. Wait 1-2 minutes and refresh portal.

If still not showing, share call time + phone number + screenshot with support.
